Welcome to api2lod.appspot.com!

API2LOD ("Happy to LOD") is a tool that facilitates exposing the content of data API onto the Web as Linked Data. This tool is available both as a downloadable servlet and an hosted service. There are several linker services which connect the data to other data sets.
